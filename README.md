# PicD

A simple Clojure application for uploading and displaying pictures. Using Noir, Monger, MongoHQ, MongoDB, GridFS.

## Usage

Prerequisite - Leiningen

lein deps
lein run


Edit your mongohq details or use a local mongodb.
